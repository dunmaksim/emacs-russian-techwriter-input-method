В отличие от метода ввода =russian-computer=, входящего в состав Emacs, этот пакет предоставляет лигатуры. Список поддерживаемых лигатур приведён ниже. Для вставки лигатуры нажмите клавишу =/= (справа от =ю=), после чего введите последовательность знаков.

** Установка

*** Добавьте в `init.el`:

#+begin_src emacs-lisp
(use-package russian-techwriter-input-method
  :ensure t
  :custom
  (default-input-method 'russian-techwriter))
#+end_src

*** Переключите метод ввода с помощью =C-\=.

** Использование

*** Лигатуры

**** Тире

| Последовательность | Знак замены | Описание    |
|--------------------+-------------+-------------|
| =-=                | –           | EN DASH     |
| =--=               | ‒           | FIGURE DASH |
| =---=              | —           | EM DASH     |

**** Кавычки

| Последовательность | Знак замены | Описание                                   |
|--------------------+-------------+--------------------------------------------|
| =/,,=              | „           | DOUBLE LOW-9 QUOTATION MARK                |
| =/,=               | ‚           | SINGLE LOW-9 QUOTATION MARK                |
| =/``=              | ‘           | LEFT SINGLE QUOTATION MARK                 |
| =/'=               | ’           | RIGHT SINGLE QUOTATION MARK                |
| =/``=              | “           | LEFT DOUBLE QUOTATION MARK                 |
| =/''=              | ”           | RIGHT DOUBLE QUOTATION MARK                |
| =/<<=              | «           | LEFT-POINTING DOUBLE ANGLE QUOTATION MARK  |
| =/>>=              | »           | RIGHT-POINTING DOUBLE ANGLE QUOTATION MARK |

**** Точки и пули

| Последовательность | Знак замены | Описание   |
|--------------------+-------------+------------|
| =/*=               | •           | Bullet     |
| =/.=               | ․           | Point      |
| =/..=              | ‥           | Двоточие   |
| =/...=             | …           | Многоточие |

**** Числа и параграфы

| Последовательность | Знак замены | Описание  |
|--------------------+-------------+-----------|
| =/&=               | §           | Paragraph |
| =/#=               | №           | Number    |
| =/no=              | №           | Number    |
